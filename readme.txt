A simple recipe plugin. 

== Description ==

It basically adds a post type for recipes to your site. You can publish recipes as standalone posts or include in your posts and pages. Organize your recipes in categories, cuisines, courses, seasons, ... Of course there are post images and all the normal wordpress post goodies for your recipes as well.

= Features =

* custom post type "recipe"
* a bunch of taxonomies (cateogories, cuisines, seasons, course). Support for custom taxonomies is planned for release 2.0
* options to also include categories and tags used from standard posts.
* user comments and pingbacks
* recipe photo using featured thumbnail tools.
* custom fields and trackbacks

= Languages =

This plugin includes the following translations:

* German
If you want a translation of another language included please help translating!

= Future Features =

* custom taxonomies
* simplified options page

== Installation ==

1. Upload `recipe-press-reloaded` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure the plugin on the Recipes menu screen.
